Lancer une fois (dépendance qui n'est plus trouvable sur la repo central Maven), mais nécessaire au build du war :mvn install:install-file -Dfile=lib/gnujaxp-1.0.0.jar -DgroupId=gnujaxp -DartifactId=gnujaxp -Dversion=1.0.0 -Dpackaging=jar -DgeneratePom=truePour packager sans lancer les tests :mvn -Dmaven.test.skip=true package
